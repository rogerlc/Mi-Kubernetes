# Create-efs.sh

## Overview

Este helper script tiene como objetivo crear un filesystem EFS en el mismo VPC donde reside nuestro cluster EKS.

## Usage

### Crear Filesystem

  ```shell
  create-efs.sh [-c | --create ] cluster_name
  ```

### Eliminar Filesystem

  ```shell
  create-efs.sh [-c | --create ] cluster_name
  ```
